Vacation account at a glance — big remaining-days number over a segmented bar (taken solid, planned hatched, rest track). Use wherever an absence balance is summarized: Absence screen, Profile, phone cards.

```jsx
<LeaveBalance entitlement={30} taken={11} planned={5} carryover={2} />
```

`label`/`unit` retitle it for other accounts (e.g. `label="Gleittage"`). Days are discrete — never render this as a ring.
